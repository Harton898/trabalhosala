Sobre este site

Uma pequena apresentação das funcionalidades, tecnologias usadas e decisões de design.

Funcionalidades principais

  • Layout responsivo e moderno usando Bootstrap 5.
  • Jumbotron estilizado com chamada principal e botão de acesso.
  • Carrossel com imagens de natureza para destaque visual.
  • Formulário de login em um modal com validação básica em JavaScript.
  • Páginas internas: content.html (explorar a natureza) e esta site.html.
  • Fundo com imagem natural + hachuras verdes para textura, garantindo contraste e atmosfera.

Como o site foi feito

O site foi desenvolvido diretamente em HTML/CSS/JS com foco em simplicidade e acessibilidade. As decisões mais importantes:

  1. Framework CSS: Bootstrap 5 para grid, componentes e responsividade.
  2. Ícones: Bootstrap Icons para elementos visuais leves.
  3. Imagens de fundo e carrossel: URLs públicas (Unsplash) para prototipagem rápida.
  4. JS leve: manipulação do formulário de login e scroll suave sem frameworks adicionais.
  5. Design: paleta verde natural, gradientes e sombras suaves para legibilidade.

Boas práticas e próximas melhorias

  • Adicionar autenticação real (backend) para o login.
  • Otimizar imagens (tamanho e formatos WebP) para performance.
  • Adicionar meta tags OG e SEO on-page para compartilhamento.
  • Melhorar acessibilidade (ARIA, contraste e navegação por teclado).
  • Implementar testes e pipeline de build para produção.

Arquivos chave
